AJAX Type Ahead

Ajax (Asynchronous JavaScript and XML) is a method of building interactive applications for the Web that process user requests immediately. ... The interface allows the user to change views and manipulate the map in real time. Ajax applications do not require installation of a plug-in, but work directly with a Web browser.

Here the AJAX method is used to sort through a large dataset of cities in USA, and while the user are entering the search shows the matches.
